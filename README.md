# 🤖 Local LLM Telegram Bot

Telegram бот, который проксирует сообщения пользователя к локальной LLM через OpenAI-compatible API.

## Фичи

- 💬 Поддержка диалогов с историей сообщений
- 📏 Авто-разбивка длинных ответов (>4000 символов)
- 🧹 Команда `/new` для сброса истории
- ⚡ Асинхронный, на `aiogram 3.x`

## Быстрый старт

### 1. Создать бота в Telegram

Откройте [@BotFather](https://t.me/BotFather), отправьте `/newbot`, получите токен.

### 2. Настроить `.env`

```bash
cp .env.example .env   # или создайте файл вручную
```

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
| `LLM_BASE_URL` | Адрес OpenAI-compatible API (по умолч.: `http://100.74.219.85:1234/v1`) |
| `LLM_API_KEY` | API-ключ для LLM (если нужен) |
| `LLM_MODEL` | Имя модели (по умолч.: `qwen3.6-35b-a3b`) |
| `AUTHORIZED_USERS` | ID пользователей, которым разрешено пользоваться ботом (через запятую: `12345,67890`) |
| `SYSTEM_PROMPT` | Системный промпт для LLM |

> ⚠️ Без добавления вашего Telegram user ID в `AUTHORIZED_USERS` бот не будет отвечать.

### 3. Запуск

```bash
uv sync          # установить зависимости (если ещё не установлены)
uv run python bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкция |
| `/new` | Очистить историю диалога |
| *(текст)* | Отправить сообщение → получить ответ LLM |

## Стек

- **aiogram** 3.x — Telegram бот фреймворк
- **openai** SDK — клиент для LLM API (OpenAI-compatible)
- **python-dotenv** — загрузка переменных из `.env`

## Структура

```
.venv/              # виртуальное окружение uv
bot.py              # основной код бота
.env                # конфиг (токен, URL LLM)
.pyproject.toml     # проект uv
README.md
```
